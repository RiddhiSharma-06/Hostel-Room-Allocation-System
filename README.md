# 🏨 Hostel Room Allocation Management System

## 📌 Overview
The Hostel Room Allocation Management System is a Java-based console application designed to automate the process of assigning rooms to students. It ensures efficient room management, avoids duplicate allocations, and provides an easy way to track room availability.

---

## 🚀 Features
- ✅ Allocate rooms to students
- 📋 View all allocated rooms
- 🔍 Check room availability
- ❌ Remove student allocation
- ⚡ Prevent duplicate room assignments

---

## 🛠️ Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner Class

---

## ▶️ How to Run

### Step 1: Compile the program
```
javac HostelRoomAllocation.java
```

### Step 2: Run the program
```
java HostelRoomAllocation
```
---

## ⚙️ How It Works
- User selects an option from the menu
- System processes input using Java logic
- Room data is stored using ArrayList
- Prevents duplicate room allocation
